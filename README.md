欢迎使用 **Govee Community Tools**！

## 1. 基本功能介绍

| 功能 | 描述 |

| ✅ 账户生成 | 自动生成符合 Govee 注册规则的测试账户 |

## 2. 启动程序

确保已安装依赖后，运行主程序：

```bash
python main.py
```

程序启动后将自动打开图形界面窗口。

> 💡 提示：推荐在虚拟环境中运行以避免依赖冲突。

## 4. 功能使用指南

### 4.1 账户管理

**功能路径**：`账户管理` → `创建新账户`

- 点击“生成账户”按钮，系统将自动生成一个随机邮箱和密码。
- 可选择保存至 `accounts_dev.json` 或 `accounts_pda.json`。
- 支持手动导入已有账户列表（JSON 格式）。

> ⚠️ 注意：请确保邮箱服务可接收 Govee 验证邮件。

---

### 4.2 邮箱验证

**功能路径**：`账户管理` → `验证邮箱`

- 输入邮箱地址或选择待验证账户。
- 点击“开始验证”，程序将轮询邮箱服务器获取验证码。
- 自动提交验证请求，更新账户状态为“已验证”。

🔧 **支持邮箱协议**：
- IMAP（Gmail、Outlook、Yahoo 等）
- 自定义邮箱服务器配置（需在 `settings.py` 中设置）

---

### 4.3 登录与会话控制

**功能路径**：`操作中心` → `登录`

- 选择已生成的账户，点击“登录”。
- 系统调用 `auth.py` 模块进行身份认证。
- 成功后自动创建会话，并显示在“当前会话”面板中。

📌 **会话有效期**：默认 2 小时（可在 `session_manager.py` 中调整）

---

### 4.4 执行社区操作

**功能路径**：`操作中心` → `执行任务`

支持以下操作：
- 发布主题帖
- 回复评论
- 点赞/取消点赞
- 收藏帖子
- 批量执行（通过任务队列）

⚙️ **任务配置**：
- 在表单中填写目标 URL、内容、延迟时间等参数
- 点击“添加到队列”并启动执行器

---

### 4.5 管理员功能

**功能路径**：`管理员` → 输入管理员密码后解锁

> 🔐 需提前在 `admin_settings.py` 中设置管理员账号和密码。

包含功能：
- 查看所有活跃会话
- 封禁异常账户
- 导出操作日志
- 清理缓存数据
- 强制注销用户会话

## 5. 配置文件说明

| 文件 | 用途 | 修改建议 |
|------|------|----------|
| `config/settings.py` | 全局设置（API 地址、超时、重试次数） | 根据网络环境调整 |
| `config/admin_settings.py` | 管理员凭证与权限设置 | 勿提交到 Git |
| `resources/accounts_dev.json` | 开发用账户池 | 用于测试 |
| `resources/accounts_pda.json` | 生产/模拟环境账户 | 注意隐私保护 |
| `data/history/` | 自动保存任务历史 | 可定期备份 |

## 6. 日志查看

所有操作均记录在：

```
logs/app.log
```

日志级别包括：
- `INFO`：常规操作
- `WARNING`：潜在问题
- `ERROR`：执行失败
- `DEBUG`：详细调试信息（需在设置中开启）

可通过 GUI 的“日志”页面实时查看。

## 7. 常见问题（FAQ）

### Q1：无法收到验证邮件？
- 检查邮箱是否正确
- 确认未被归入垃圾邮件
- 检查 IMAP 是否启用（如 Gmail 需开启“应用专用密码”）

### Q2：登录失败？
- 确保账户已验证
- 检查 Govee 是否有 IP 限制
- 尝试更换 User-Agent 或代理

### Q3：GUI 启动黑屏或无响应？
- 确保 `tkinter` 已安装（Python 标准库）
- 尝试运行 `python -m tkinter` 测试 GUI 支持
- 更新显卡驱动或使用无头模式调试

### Q4：如何更新工具版本？
- 拉取最新代码
- 运行 `pip install -r requirements.txt --upgrade`
- 检查 `config/_version_.py` 是否更新
